# The documnetation of Languange Engine

Language Engine is a header only library that provide a simple solution to multilanguage application.

Language Engine's translation files are written in JSON format and using [rapidJSON](http://rapidjson.org) as the phraser.

## Compile Time Configurations

This library provide a "compile time configuration" to control how this library to be compile.  
Refer [here](PrecompileConfig.md) for more information.

## About String Path
Language Engine use String Path to locate translated string. Visit [here](StringPath.md) for more information.
